# BuildVan
Getting the data
```
- To create an update or new set of data 
    - create a python environment and run pip install -r requirement.txt
    - download property-parcel-polygon dataset from opendata.vancouver.ca
    - run exec.py
- The collated data (Jun 30, 2025) is located at ./data/processed_data.json
- Make sure the data is located in ./data folder
```
The website
```
- To run your own website use google maps API 
- Add it into the HTML body section
- Run the HTML
```
